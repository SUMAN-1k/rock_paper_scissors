# ✊✋✌️ Rock-Paper-Scissors with Text-to-Speech | Python

This is a fun and interactive **Rock-Paper-Scissors** game created using Python. The twist? The game **talks back** to you using **Text-to-Speech (TTS)**!

---

## 🔧 Features

- Classic Rock-Paper-Scissors game logic
- Text-to-Speech conversion for game instructions and results
- Randomized computer choice for dynamic gameplay
- Clear terminal-based interface

---

## 🗣️ Libraries Used

- `random` – to randomize computer's choice  
- `pyttsx3` – for offline text-to-speech functionality  

Install dependencies using pip:

```bash
pip install pyttsx3
